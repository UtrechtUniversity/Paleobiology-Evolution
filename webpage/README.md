# Paleobiology & Evolution computer practicals

Quarto website for the Paleobiology & Evolution course practicals.

## Structure

- `index.qmd` - Homepage 
- `_quarto.yml` - Website configuration file
- `practicals/` - Directory containing all practicals
- `styles.css`
- `.nojekyll` - Tells GitHub Pages not to use Jekyll processing

## Local Development

To render the website:

```bash
cd webpage
quarto render
```
